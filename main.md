# Заголовок

## Научились коммитеть - создавать комментарии. 

```
git commit -m
```

## Научились узнавать статус.

```
git status
```

## Научились узнавать на какой ветке мы.

```
git branch
```

## Научились удалять ветвь.

```
git branch -d <branch>
```

## Научились менять названия ветвей.

```
git branch -m <old_branch> <new_branch>
```
